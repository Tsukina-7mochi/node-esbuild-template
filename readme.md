# Building browser content with esbuild

## Usage

- build
  - production
    ```shell
    $npm run build
    ```
  - development
    ```shell
    $npm run dev
    ```
  - development with watch
    ```shell
    $npm run watch
    ```
- CI
  - lint
    ```shell
    $npm run lint
    ```
  - format
    ```
    $npm run fmt
    ```
  - format check
    ```
    $npm run fmt-check
    ```
